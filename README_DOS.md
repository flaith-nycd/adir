# ADIR - Apple Disk Image Reader

Dump track/sector of an Apple DOS disk image file (dsk)

## How-to

read_ts.py [FILENAME] [TRACK] [SECTOR] [SIZE]
- FILENAME: Disk filename
- TRACK: Which track (default = 17 - $11)
- SECTOR: Which sector (default = 0 - $00)
- SIZE: How many bytes to dump (default = 256 - $FF)
