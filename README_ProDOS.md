# ADIR - Apple Disk Image Reader

Dump a block of a ProDOS disk image file (dsk, po)

## How-to

read_block.py [FILENAME] [BLOCK]
- FILENAME: Disk filename
- BLOCK: Which block (default = 0 - $00)
```
python read_block.py ProDOS_2_0_3.dsk
```
